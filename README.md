# 🩺 RxVision

**RxVision** is an AI-powered prescription scanner that helps users digitize and understand their medical prescriptions. By leveraging OCR technology and Gemini AI, it extracts structured information from images to help you manage your health safely.

> **⚠️ Disclaimer:** This application is for informational purposes only and does not provide professional medical advice. Always consult with a doctor or pharmacist regarding your medications.

---

## 🚀 Features
* **OCR Scanning:** Digitize handwritten or printed prescriptions using camera capture or file uploads.
* **AI-Powered Extraction:** Automatically parses medical data into a clean, structured format (Patient Name, Doctor, Medications, Dosage).
* **Medical Guide Assistant:** An interactive assistant to answer questions about the extracted medications.
* **Privacy-Focused:** Designed with a local-first approach to data processing.

## 🛠️ Tech Stack
| Component | Technology |

## 💡 Usage Instructions
* Navigate to the Scan Prescription section.

* Upload an image or use your camera to take a clear photo of the prescription.

* Click Read Prescription to extract the data.

* Use the Medical Guide Assistant box to ask follow-up questions about the findings.
